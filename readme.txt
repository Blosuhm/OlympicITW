# OlympicITW ( index.html para inciar na página inicial )

Opós a apresentação:
    - Adcionamos um mapa na página "Games" com todos os jogos distinguidos por "Summer" e "Winter". E que ao clicar no marcador direciona para os detalhes do respetivo Jogo.
    - Adionamos um Accordion na página "Favoritos", e quando não existem favoritos de uma determinada categoria ele nem sequer cria o "accordion-item"

Nota importante:
    - Na página countries.html as medalhas de cada país só são exibidas ao abrir o site em um "Servidor", pois tivemos de adicionar um arquivo json (medalhas.json). Pois a API em "http://192.168.160.58/Olympics/api/Statistics/Medals_Country" está com algum problema e demora pelo menos 5 segundos a abrir o que não era viável. Na nossa apresentação mostramos que esta funcionalidade funcionava!
